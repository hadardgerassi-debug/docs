title: Troubleshooting description: Fix common issues quickly
Common issues and quick fixes

Sidebar item not showing

Make sure the path is listed in docs.json and matches the filename (no .mdx).
Page shows “Not found”

Check for typos in the file path or name in docs.json.
Changes not appearing

Confirm you clicked “Commit changes” for each edit, then wait ~30–60 seconds and refresh.
